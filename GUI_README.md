# 🎵 Spotify Music Downloader

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 2. Запуск приложения

```bash
python gui_app.py
```

или

```bash
python3 gui_app.py
```

## 📖 Использование

### Первый запуск

1. **Запустите приложение** - откроется главное окно
2. **Перейдите во вкладку "⚙️ Настройки"**
3. **Заполните настройки Spotify API:**
   - **Client ID** - получите на [Spotify Developer Dashboard](https://developer.spotify.com/dashboard)
   - **Client Secret** - получите там же
   - **Spotify Username** - ваш username из профиля Spotify
   - **Папка для скачивания** - выберите папку, куда сохранять музыку
   - **Задержка между запросами** - рекомендуется 1-2 секунды
4. **Нажмите "💾 Сохранить настройки"**

### Скачивание плейлистов

1. **Перейдите во вкладку "📥 Скачивание"**
2. **Вставьте URL плейлиста** в поле ввода
3. **Нажмите "➕ Добавить URL"** - URL добавится в список
4. **Повторите для других плейлистов** (если нужно)
5. **Нажмите "🚀 Начать скачивание"**
6. **Следите за прогрессом** во вкладке "📋 Логи"

### Управление списком URL

- **➕ Добавить URL** - добавляет URL из поля ввода в список
- **🗑️ Очистить** - очищает весь список
- **➖ Удалить выбранное** - удаляет выделенный URL из списка

## 🎯 Примеры использования

### Скачать один плейлист

1. Вставьте URL: `https://open.spotify.com/playlist/4DmELrQMOuWomBtcauF4g0?si=...`
2. Нажмите "➕ Добавить URL"
3. Нажмите "🚀 Начать скачивание"

### Скачать несколько плейлистов

1. Добавьте первый URL и нажмите "➕ Добавить URL"
2. Добавьте второй URL и нажмите "➕ Добавить URL"
3. Повторите для всех нужных плейлистов
4. Нажмите "🚀 Начать скачивание"

## ⚙️ Настройки

### Spotify API

Для работы приложения нужны ключи Spotify API:

1. Перейдите на [Spotify Developer Dashboard](https://developer.spotify.com/dashboard)
2. Войдите в свой аккаунт Spotify
3. Нажмите "Create App"
4. Заполните форму:
   - **App name**: любое имя (например, "Music Downloader")
   - **App description**: описание приложения
   - **Redirect URI**: `http://localhost:8888/callback`
5. Скопируйте **Client ID** и **Client Secret**

### Spotify Username

Как найти свой Spotify username:
1. Откройте Spotify в браузере
2. Перейдите в свой профиль
3. Скопируйте username из URL (например, если URL `https://open.spotify.com/user/username123`, то username = `username123`)
